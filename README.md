# mapsi
(Map-Tax) Taxi path finding on a city map visualizer.


## Project Setup
```bash
npm install
```

#### Compiles and hot-reloads for development
```bash
npm run serve
```

#### Compiles and minifies for production
```bash
npm run build
```

#### Lints and fixes files
```bash
npm run lint
```
